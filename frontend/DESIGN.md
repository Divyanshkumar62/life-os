# Life-OS Design System

This design system is derived from the project's `tailwind.config.js` and global styles. It defines the visual language for the "Life-OS" application.

## Core Theme
**Style:** Dark Futuristic / Cyberpunk Lite
**Visuals:** Neon glows, deep backgrounds, high contrast.

## 1. Color Palette

### Backgrounds & Surfaces
| Use Case | Color Name | Hex | Description |
| :--- | :--- | :--- | :--- |
| **App Background** | `gray-900` | `#111111` | Deepest black/gray for main background |
| **Card Surface** | `gray-800` | `#1a1a1a` | Slightly lighter for cards/containers |
| **Secondary Surface** | `gray-700` | `#2a2a2a` | Modals, dropdowns, inputs |

### Primary Accents (Cyan)
Used for primary actions, active states, and focus rings.
- **Base:** `cyan-500` (#06b6d4)
- **Glow:** `cyan-400` (#22d3ee)
- **Hover:** `cyan-600` (#0891b2)

### Secondary Accents (Teal & Blue)
Used for distinct indicators, secondary buttons, or data visualization.
- **Teal:** `teal-500` (#14b8a6)
- **Blue:** `blue-500` (#3b82f6)

### Functional Colors
- **Success:** `#10b981` (Green)
- **Warning:** `#f59e0b` (Amber)
- **Error:** `#ef4444` (Red)
- **Info:** `#3b82f6` (Blue)

## 2. Typography
- **Primary Font:** Inter, system-ui, sans-serif
- **Monospace:** JetBrains Mono, Courier New (for code/data)

## 3. UI Components

### Buttons
- **Shape:** Rounded (8px radius)
- **Padding:** 0.6em 1.2em
- **Background:** `gray-800` (`#1a1a1a`) with `1px` transparent border
- **Hover:** Border color changes to `#646cff` (or Primary Cyan in updated designs)
- **Focus:** Standard focus ring

### Cards
- **Shadow:** `shadow-card` (0 4px 6px -1px rgba(0, 0, 0, 0.3))
- **Padding:** 2em
- **Background:** `gray-800`

### Effects & Animations
- **Neon Glow:** `shadow-glow-cyan` (0 0 20px rgba(34, 211, 238, 0.5)) - Use for "active" or "powered up" items.
- **Pulse:** `animate-pulse-glow` - For alerts or notifications.
- **Slide In:** `animate-slide-in` - For entering elements.

## 4. Layout
- **Max Width:** 1280px (centered)
- **Padding:** 2rem generic container padding
